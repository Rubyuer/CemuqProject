# 牛肉鲜生

  应用介绍:"牛肉鲜生"app是一款线上购买牛肉干、牛排的应用。
  
  应用功能:
    
    1.商品分类：牛肉干、牛排
      牛肉干：裹汁牛肉、瓦块牛肉、香辣牛肉、以及牛肉干礼盒装；
      牛排：原切牛排、菲力/肉眼/T骨/西冷牛排等。
      
    2.积分商城：积分换商品，积分通过订单完成获得对应积分数量。
    
    3.购物车："牛肉鲜生"全场商品均可加入购物车进行一次性结算，一次配送。
    
    4.地址：我的地址，新增地址、修改地址，下单时选择商品配送的最终地址。
    
    5.订单：全部订单、待发货订单、已发货订单、已收货订单、已取消订单等。
    
    6.其他：优惠券、我的积分、我的收藏、相关设置等。
